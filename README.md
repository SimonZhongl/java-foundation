一些自己的Java 练习题目
---------------
【程序1】 <br> 
题目：古典问题：有一对兔子，从出生后第3个月起每个月都生一对兔子，小兔子长到第三个月后每个月又生一对兔子，假如兔子都不死，问每个月的兔子总数为多少？ <br> 
1.程序分析： 兔子的规律为数列1,1,2,3,5,8,13,21.... <br> 
<br> 
【程序2】 <br> 
题目：判断101-200之间有多少个素数，并输出所有素数。 <br>
1.程序分析：判断素数的方法：用一个数分别去除2到sqrt(这个数)，如果能被整除， <br> 
则表明此数不是素数，反之是素数。 <br>
